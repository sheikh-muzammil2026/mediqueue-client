# 📌 Project Overview

MediQueue is a modern Tutor Booking web application where students can browse tutors, explore tutor details, and book learning sessions easily.

The platform simplifies tutor scheduling by managing session availability, booking status, and session organization efficiently.

---

# 🚀 Features

- 🔐 Authentication System using Better Auth
- 🌙 Dark & Light Theme Toggle
- 📚 Tutor Booking & Session Management
- 🔍 Tutor Search & Date Filter System
- 🧾 JWT Protected Private Routes
- ✏️ Add, Update & Delete Tutor Features
- 📅 Session Date Validation & Slot Control
- 📱 Fully Responsive for Mobile, Tablet & Desktop
- 🔔 Beautiful Toast Notifications for CRUD Operations
- ⚡ Dynamic Page Title & Loading Spinner
- ❌ Custom 404 Not Found Page
